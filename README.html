Solid Parse

REQUIREMENTS:

  • Python 3+ [Installed To PATH]

USAGE:

  • Copy the target file and paste it into the Mount Bar
  • The program will not accept the whole file but will only take its address or location
  • If you followed instructions, you'll see a message that says 'mounted'
  • You can now search for anything in the file.
  • Click on 'Refresh' If you want to clear the output
  • You can click 'File' then 'Print' (to view in notepad) or 'Print to Browser' To view in browser.
  • Supported File Formats: .txt, .ini, .csv, .xml, .py, .java, etc

    • Currently, Solid Parse is mostly used for retrieving hidden direct download links from websites including specific file locations on servers which then allows the user to download for example: a video, from an otherwise 'stream only site'.

    This software can work with very large text files. A lag may occur when handling such files on low end hardware.
    This product comes with a text file containing more than 16 000 video game records. The name of the file is "vgsales.txt"
    You can mount it and search for any popular video game to see how the product works e.g. Need For Speed

    TIP: You can narrow down your search by using specific keywords. For example: Battlefield 4, instead of Battlefield. To retrieve the entire text found in a file, just click on Mount a file, and click on search without typing anything in the search bar

    STRING MANIPULATION

    BUTTONS AND COMMANDS

  • RC
    • Remove Comma - This removes commas from the file before outputting the contents to display
  • RW
    • Remove Whitespace - This removes whitespaces from the file before outputting the contents to display
  • GL
    • Get Lowercase - This allows you to search without worrying about case sensitivity.
  • GR
    • Get Range - This tells you how many characters and lines of code a file contains
